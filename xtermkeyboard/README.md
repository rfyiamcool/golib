# xtermkeyboard

识别在线终端xterm.js的键盘输入.

- keyboardCodeDelete = "7f"
- keyboardCodeEnter  = "0d"
- keyboardCodeSpace  = "20"
- keyboardCodeTab    = "09"
- keyboardCodeEsc    = "1b"
- keyboardCodeUp     = "1b5b41"
- keyboardCodeDown   = "1b5b42"
- keyboardCodeRight  = "1b5b43"
- keyboardCodeLeft   = "1b5b44"
